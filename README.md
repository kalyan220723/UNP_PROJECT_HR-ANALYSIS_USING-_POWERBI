# UNP_PROJECT_HR-ANALYSIS_USING-_POWERBI

I did this project in collaboration with UNP, where I developed an interactive HR Attrition Analytics dashboard using Power BI to analyze employee turnover patterns and uncover key factors driving attrition.

The project involved data cleaning, transformation, and modeling to create a structured dataset suitable for analysis. I designed intuitive and visually consistent dashboards that provide insights into attrition trends based on factors such as department, job role, age group, salary, and years at the company.

Key features of the project include:

Identification of high-risk employee segments prone to attrition

KPI tracking such as overall attrition rate, average tenure, and employee demographics

Use of slicers and filters for dynamic, user-driven exploration

Implementation of DAX measures for custom calculations and deeper insights

This project demonstrates my ability to translate raw HR data into actionable business insights, enabling organizations to make data-driven decisions to improve employee retention and workforce planning.
